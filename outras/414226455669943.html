"Até o início desse ano eu trabalhava como administador de rede, e vendo todo o tráfego da rede, só é possível perceber que se está usando o tor, a request http é encriptada. A única possibilidade pra man in the middle por exit nodes do tor. O único jeito de backtrack é se alguém possui vários nós do tor. A definição de exploit é: um método de obter vantagem e se explorar uma falha de uma tecnologia. Não é ""minha lógica "", é assim que é definido a estrutura de rede. Pegar um servidor e fazer honeypot não é exploit, não tem nada a ver com a camada do tor, é independente do protocolo, ou seja, não importa se você usa tor ou não, se alguém pega o servidor pode acontecer isso, mas os dados que eles pegam são os que você coloca no servidor deles. Exploit é, através de alguma falha de segurança (por ex no flash) é possível criar uma conexão direta entre o host e o cara que fez o exploit, deanomizando a pessoa, e possivelmente pegando dados que NAO foram colocados no servidor do site. É assim que funciona, foi assim que o FBI conseguiu obter a localização de vários usuários de sites com conteúdo ilegal, não era só porque pegaram o site, eles fizeram um exploit em vulnerabilidade do Flash..."
